# EDR-KE-010: Knowledge Fragment Identity

Status: Deferred

Target:
Document Engine and fragment-promotion slice

Decision:
Knowledge fragments are now defined conceptually as revision-bound, locator-addressable units, but their durable identity model is deferred until fragment candidates and authoritative fragment promotion are implemented.

Required future clarification:

- when a fragment receives a stable ID
- whether fragment identity survives equivalent locator changes across revisions
- how fragment-level supersession is represented
- how stale locators are preserved historically
